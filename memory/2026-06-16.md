# 2026-06-16 记忆

## 《404》游戏项目 — AI全流程开发实验

### 参与者
- 印子姐：制作人/创意总监 — 定方向、验收关卡设计
- 磊子：AI管线调度员+技术兜底 — 遇到管线走不通的地方手动补

### 游戏核心概念
**《404》** — Roguelike叙事卡牌解谜游戏
- 你被花盆砸死，在阴间街道办事处当临时信差#404
- 送完99封信（简化版36个灵魂故事）才能投胎
- 非战斗卡牌系统：倾听、重构、释放、连接、荒诞
- 核心卖点：笑着笑着就哭了

### 三幕剧结构
- 第一幕「凡所有相」— 荒诞喜剧（毛肚哥→包租公）
- 第二幕「皆是虚妄」— 笑中带泪
- 第三幕「若見諸相非相，即見如來」— 深海，最终通关唤醒

### 核心玩法（简版确认）
- 进入记忆房间→叙事（视觉小说风）
- 房间内有解谜→代表记忆扭曲点
- 卡牌重构记忆（不是攻击，是理解/对话/重新诠释）
- 解开→新视角/卡牌/剧情进展
- 卡住→消耗共情值（非血量），换新理解卡
- 走到终点→灵魂释然，玩家也变了

### 场景递进叙事方案
- 不采用实时NPC跟随（减少开发量）
- 改为"进入房间→触发对话→解谜→进入下一房间→NPC已在里面等你"
- 走廊灯箱字幕等效果用场景入场动画实现

### 技术选型（已定）
- 引擎：Godot 4（GDScript）
- 叙事：Ink / Yarn Spinner
- AI管线：LLM生成叙事分支+谜题 / SD出美术概念图 / Suno出音乐
- 美术风格：低保真+高情感密度，水彩风/手绘线条，灰→暖渐变

### 毛肚哥关卡 - 首个测试用例
- 6个房间：包间(起点)→走廊(探索)→大厅(碎片收集)→后厨(谜题)→包间·回忆(叙事核心)→天台(决策点)
- 表层故事：因一块毛肚被老婆用筷子戳死
- 内核：老友大刘的友情，绝交后最后一顿饭
- 情绪节奏：好笑→好奇→渐冷→专注→戳心但暖→释然
- 预计通关时间20-30min
- 后续将作为AI管线第一步跑通的原型关卡
