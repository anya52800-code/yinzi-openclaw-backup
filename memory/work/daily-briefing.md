## 每日简报推送服务（正式版）

### 推送设置
- **时间**: 每日上午 11:00
- **形式**: 飞书消息推送
- **语言**: 中文

### 内容板块

| 板块 | 条数 | 内容类型 |
|------|:----:|----------|
| 🔬 科技 | 4-5条 | AI、硬件、行业动态 |
| 💹 经济 | 3-4条 | 市场、政策、产业动态 |
| 🎮 游戏 | 2-3条 | 游戏发布、设备、行业 |
| 📚 中文图书 | 12本 | 艺术3+成长3+小说3+传记3 |

### 排版格式

**标题**:
```
# 📰 每日简报 · [日期]
```

**板块标题**:
```
## [emoji] [类别名]
```

**新闻表格**（科技/经济/游戏）:
```
| 新闻 | 摘要 | 链接 |
|------|------|------|
| **标题** | 一句话摘要 | [查看详情](URL) |
```

**图书列表**（4个类别）:
```
**[emoji] 类别名**
- [《书名》](URL) 作者 ⭐评分 | 一句话推荐
```

**底部信息**:
```
⏰ 每日11:00推送 | 💬 回复"更多+类别"获取详情
```

### 信息来源

**科技**:
- The Verge (https://www.theverge.com)
- TechCrunch
- 36氪

**经济**:
- Bloomberg (https://www.bloomberg.com/asia)
- Financial Times
- 财新

**游戏**:
- Variety (https://variety.com)
- 游戏葡萄

**中文图书**:
- 豆瓣读书 (https://book.douban.com)
- 微信读书
- 京东图书畅销榜
- 当当网

### 内容筛选逻辑

1. **时效性**: 24小时内发布
2. **重要性**: 行业重大事件、权威发布
3. **可读性**: 一句话能概括核心内容
4. **链接**: 每条新闻/图书附直达链接
