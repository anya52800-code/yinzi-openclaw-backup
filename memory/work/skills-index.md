## 🎯 创作前期分析（新增）

| 技能 | 核心用途 | 星级 |
|------|----------|:----:|
| **trend-analyzer** | 热点追踪、趋势预测、关键词研究、最佳发布时机 | ⭐⭐⭐⭐⭐ |
| **content-strategy** | 受众分析、内容定位、pillar设计、发布策略 | ⭐⭐⭐⭐⭐ |
| **research-assistant** | 信息收集、资料整理、研究报告、洞察提炼 | ⭐⭐⭐⭐⭐ |
| **competitive-analysis** | 竞品拆解、差距分析、机会识别、差异化定位 | ⭐⭐⭐⭐ |

**创作前期工作流**：
1. **趋势分析** → trend-analyzer（找热点/选时机）
2. **竞品研究** → competitive-analysis（看对手/找机会）
3. **信息调研** → research-assistant（收资料/出洞察）
4. **策略制定** → content-strategy（定受众/排内容）
5. **内容执行** → article-writer/remotion/etc（创作产出）

## 📦 备份与归档

### 自动备份系统
- **脚本位置**: `scripts/`
- **备份内容**: memory/, MEMORY.md, AGENTS.md, skills索引
- **执行频率**: 
  - 每日 21:00 - Git备份
  - 每周五 21:30 - 周归档
- **归档位置**: `backup/weekly/`, `backup/monthly/`

### 手动执行
```powershell
# 快速备份
.\scripts\backup-daily.ps1

# 完整流程（Git + 归档 + 飞书准备）
.\scripts\backup-full.ps1

# 强制归档（非周五执行）
.\scripts\backup-full.ps1 --archive
```
