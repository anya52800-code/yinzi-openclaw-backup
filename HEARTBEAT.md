# HEARTBEAT.md

## 每日固定提醒任务（自动执行）

### Session Start（每次启动）
1. ✅ **记忆系统检查**
   - 检查/创建当日记忆文件 `memory/YYYY-MM-DD.md`
   - 回顾昨日记忆，确认无遗漏
   - 清理 BOOTSTRAP.md（如存在）

2. ✅ **主动记忆检索与关联**（新增）
   - 读取今日记忆文件，获取当前上下文
   - 使用 memory_search 搜索相关历史记录
   - 提取关键信息：项目名称、人物、技能、待办
   - **自动关联**：发现跨项目联系时主动提及
   - 将关联结果带入当前会话上下文

2. ✅ **技能系统检查**
   - 验证46个核心技能完整性
   - 确认21个5星技能可用
   - 检查代码验证器90分门禁

3. ✅ **工作区检查**
   - 查看昨日待办完成情况
   - 更新技能索引（如有变更）
   - 同步重要决策到MEMORY.md

### Heartbeat Check（每30分钟）
- [ ] 检查是否有紧急提醒触发
- [ ] 查看待办事项进度
- [ ] 备份关键记忆文件
