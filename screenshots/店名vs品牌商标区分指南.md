# 店名 vs 品牌商标 — 区分指南

## 一句话结论

| | 店名（营业执照） | 品牌商标 |
|--|----------------|---------|
| **管什么** | 你在工商局登记的"字号" | 你对某个名称的**全国专用权** |
| **保护范围** | 你登记的那个区县，同行业不重名 | **全国**，你注册的类别 |
| **能不能阻止别人用** | ❌ 不能 | ✅ 能 |
| **被侵权了怎么办** | 基本没办法 | 可以起诉、投诉平台下架 |

## 什么时候必须注册商标？

| 场景 | 建议 |
|------|------|
| **想长期做这个品牌** | 必须注册，否则就是给别人养孩子 |
| **有线上销售**（淘宝/抖音/小程序） | 必须注册，平台要求商标才能开旗舰店 |
| **想做加盟/连锁** | 必须注册，没商标没法授权别人 |
| **产品要进商超/渠道** | 必须注册，渠道商要看商标证书 |
| **只在本地开一家小茶馆** | 可以缓，但建议注册（成本低，防抢注） |

## 注册哪个？店名还是品牌名？

| 策略 | 操作 | 适合谁 |
|------|------|--------|
| **店名=品牌名** | "朴茶别院"既做店名，也注册商标 | 单品牌、长期做 |
| **店名≠品牌名** | 店名随便叫，品牌另外注册（如"一盏朴茶"） | 多店、多品牌、灵活调整 |
| **店名+图形商标** | "朴茶别院"注册不了，注册logo图形 | 名字被抢注，用视觉识别 |

## 不注册的风险

| 风险 | 真实案例 |
|------|---------|
| **被抢注** | 你做了3年，有人把你名字注册了，反咬你侵权 |
| **平台下架** | 抖音/淘宝有人投诉你"无商标售卖"，链接被下 |
| **做不了加盟** | 想扩张时发现没商标，没法授权 |
| **品牌价值归零** | 投入了装修、宣传、口碑，最后名字不是自己的 |
