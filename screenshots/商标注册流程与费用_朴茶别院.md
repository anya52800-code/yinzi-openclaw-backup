# 商标注册流程与费用（中国大陆）

## 一、注册流程

```
申请提交 → 形式审查（1个月）→ 实质审查（4-6个月）→ 初审公告（3个月）→ 注册公告 → 拿证
```

| 阶段 | 时间 | 内容 |
|------|------|------|
| **准备材料** | 1-3天 | 确定商标、类别、申请人信息 |
| **提交申请** | 当天 | 网上提交或代理提交 |
| **形式审查** | 约1个月 | 检查材料是否齐全、格式是否正确 |
| **实质审查** | 4-6个月 | 审查是否与他人商标冲突、是否具有显著性 |
| **初审公告** | 3个月 | 公告期，他人可提异议 |
| **注册公告** | 公告期满无异议 | 下发商标注册证 |

**总时长：约9-12个月**（顺利情况下）

---

## 二、费用

### 官方费用（国家知识产权局收取）

| 项目 | 费用 |
|------|------|
| **受理商标注册费** | 300元/类（限10个商品/服务项目） |
| 超出10项 | 每项加收30元 |

### 代理费用（如委托代理机构）

| 服务类型 | 费用范围 |
|---------|---------|
| **基础代理** | 500-800元/类（含官方费） |
| **全包服务** | 1000-1500元/类（含查询、申请、跟进） |
| **加急/担保注册** | 2000元+/类（不通过退部分费用） |

### 你的情况估算

**方案A：自己申请（省钱但费事）**
- 第30类（茶叶）：300元
- 第35类（销售服务）：300元
- 第43类（茶馆服务）：300元
- **合计：900元**

**方案B：找代理（推荐）**
- 3类全包：约1500-2500元
- 含前期查询、材料准备、全程跟进

## 三、需要准备的材料

| 材料 | 说明 |
|------|------|
| **商标图样** | 文字商标直接提供文字；图形商标提供高清图 |
| **申请人身份证明** | 个人：身份证复印件；公司：营业执照复印件 |
| **商品/服务清单** | 确定每类要保护的具体项目 |
| **委托书** | 如找代理，需签字/盖章 |

---

## 四、建议注册的类别

| 类别 | 内容 | 优先级 |
|------|------|--------|
| **第30类** | 茶、茶叶、茶饮料、袋泡茶 | ⭐⭐⭐ 必注 |
| **第35类** | 茶叶零售/批发、广告宣传、电商 | ⭐⭐⭐ 必注 |
| **第43类** | 茶馆、茶艺馆、餐饮服务 | ⭐⭐ 如有实体店则注 |
| 第21类 | 茶具、茶杯、茶壶 | ⭐ 如有自有产品则注 |
| 第40类 | 茶叶加工服务 | ⭐ 如有加工业务则注 |

---

## 五、注意事项

1. **先申请原则**：同一天申请，谁先提交谁优先
2. **盲期风险**：最近1-2个月提交的申请查不到，存在撞车可能
3. **被驳回怎么办**：
   - 可申请复审（750元）
   - 或换名字重新申请
4. **有效期**：10年，到期前12个月可续展

---

## 六、推荐操作步骤

1. **确定商标名**：从"朴茶山房""璞茶别院"中选一个主攻
2. **找代理做详细查询**：确认无近似冲突（费用约200-500元）
3. **提交申请**：建议第30+35+43类一起申
4. **等待审查**：期间可正常使用™标识
5. **拿证后**：使用®标识，受法律保护

---

## 七、省钱技巧

| 技巧 | 说明 |
|------|------|
| **自己申请** | 省代理费，但需自己跟进、处理补正 |
| **先注核心类** | 第30类先拿下，其他类后面补 |
| **找本地代理** | 比大所便宜，沟通方便 |
| **关注官费减免** | 个人/小微企业有时有减免政策 |

---

*注：费用以2025年标准为准，具体以商标局最新公告为准。*
