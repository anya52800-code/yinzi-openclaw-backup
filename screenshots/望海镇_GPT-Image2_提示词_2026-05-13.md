## 五、GPT Image-2 使用技巧

| 技巧 | 说明 |
|------|------|
| **风格前缀放最前** | 前5行决定整体基调，必须每图都带 |
| **强调"minimal noise"** | GPT Image-2 容易出噪点，反复提醒 |
| **"clean flat color blocks"** | 引导AI用色块而非细碎纹理 |
| **"smooth gradients"** | 避免生硬的色彩过渡 |
| **"no gritty textures"** | 拒绝沙砾、粗糙质感 |
| **配色用英文描述+色值** | 双重保险，确保颜色准确 |
| **"breathing space"** | 引导构图留白，不要太满 |
| **先出1张确定风格** | 满意后以此为参考垫图 |

---

## 六、推荐生成顺序

1. **先做 bg_teahouse.png** — 最核心场景，确定主色调
2. **再做 bg_apartment.png** — 验证暖光表现
3. **然后角色** — 用茶馆图做风格参考 (image prompt)
4. **最后 UI** — 提取主色做按钮
