# 飞书第二个机器人申请指南

## 前提条件
- 飞书管理员权限（或找公司IT申请）
- 已有第一个机器人的经验

---

## 申请步骤

### 1. 进入飞书开放平台
- 网址: https://open.feishu.cn/
- 用公司飞书账号登录

### 2. 创建新应用
```
点击「创建应用」→「自定义应用」
应用名称: OpenClaw-Home（或你喜欢的名字）
应用描述: 家中电脑使用的AI助手
```

### 3. 获取凭证
```
进入「凭证与基础信息」:
- App ID: 复制保存
- App Secret: 点击显示，复制保存
```

### 4. 启用机器人能力
```
进入「添加应用能力」→「机器人」→ 启用
```

### 5. 配置权限
```
进入「权限管理」，添加以下权限:
- im:message:send（发送消息）
- im:message:receive（接收消息）
- im:message.group（群组消息）
- im:message.p2p（单聊消息）
```

### 6. 发布应用
```
进入「版本管理与发布」→「创建版本」
版本号: 1.0.0
更新说明: 初始版本
点击「申请发布」→ 等待管理员审批
```

### 7. 获取机器人Token
```
发布成功后:
进入「凭证与基础信息」
查看「Bot Token」（机器人令牌）
复制保存
```

---

## OpenClaw配置

### 修改配置文件
文件位置: `%USERPROFILE%\.openclaw\config.yaml`

```yaml
channels:
  feishu-home:  # 新名字，区分公司的
    kind: feishu
    appId: "新App ID"
    appSecret: "新App Secret"
    botToken: "新Bot Token"
    # 其他配置复制公司的即可
```

---

## 注意事项

| 问题 | 解决方案 |
|------|----------|
| 没有管理员权限 | 找公司IT或飞书管理员帮忙审批 |
| 公司不允许自建应用 | 用个人飞书账号创建（功能可能受限） |
| 两个机器人同时在线 | 消息互不干扰，记忆通过OneDrive同步 |
