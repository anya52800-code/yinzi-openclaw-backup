# 茶业合作备忘

## 合作备忘（发给对方确认）

[朋友名字]，确认一下咱们的合作方式：

1. 我先放一批茶和茶具在你那儿，我给你底价，统一售价我定
2. 你按统一售价卖，差价归你；私域你想卖多少自己定，多赚的也归你
3. 月结，月底对账，7天内按底价给我货款——我这边也要给上游结款，咱们节奏一致
4. 卖不完的货，随时能退我，退货物流费我出

你看看有没有问题？没问题咱们就按这个来。

## 合作要点细则

| 项目 | 内容 |
|------|------|
| 合作模式 | 代销，货权归我方 |
| 首批货值 | 不超过1万元 |
| 产品类型 | 茶叶 + 茶具（耐存放为主） |
| 定价方式 | 我方给底价+统一售价，她按统一售价卖赚差价；私域加价归她 |
| 结算周期 | 月结（月底对账，7天内按底价给我货款） |
| 对账时间 | 每月1号核对上月销售 |
| 退货机制 | 未售出产品随时可退，物流费我方承担 |
| 补货机制 | 她卖完提前3天说，我安排发货；我想补货先问她库存 |
| 新品上架 | 双方商量，她同意再寄 |
| 货损责任 | 对方保管不善造成的损坏，按底价赔偿 |
